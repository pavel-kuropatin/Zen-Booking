  Букинг:

- Зарегистрированный пользователь может снимать жильё и/или сдавать его.
- Съём доступен сразу после регистрации.
- Клиенты могут снимать апартаменты на определённое число дней, бронировать апартаменты,
  а также отказываться от брони, если дата съёма не наступила.
- История заказов каждого клиента сохраняется и доступна им для просмотра.
- Для съёма жилья необходимо выбрать предложение собственности, выбрать доступную дату, оставить заказ и ждать пока хост примет его.
- Хост может отклонить заказ.

- Для сдачи жилья нужно добавить его на свой аккаунт и добавить минимум одно изображение.
- Можно добавить фотографии жилья или изменить их в любое время.
- Жильё и фотографии должны быть одобрены модератором перед добавлением в листинг.
- В любое время владелец может убрать собственность из листинга без удаления и вернуть обратно.
- При получении заказа владелец может принять его или отклонить.

  Администрирование:

- Роль ADMIN существует изначально.
- Роль MODER может создать и банить только ADMIN.
- Модераторы должны подтверждать собственность и картинки, банить и прочее.
  Планируется:
- Администратор может простатривать действия модераторов.
- Модераторы могут просматривать историю своих действий
