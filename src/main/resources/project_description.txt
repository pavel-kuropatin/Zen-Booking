  Букинг:

- Зарегистрированный пользователь может снимать жильё и/или сдавать его.
- Съём доступен сразу после регистрации.
- Клиенты могут снимать апартаменты на определённое число дней, бронировать, а также отказываться от брони, если дата съёма не наступила.
- История заказов каждого клиента сохраняется и доступна им для просмотра.
- Для съёма жилья необходимо выбрать предложение собственности, выбрать доступную дату, оставить заказ и ждать пока хост примет его.
- Хост может отклонить заказ.
- После совершения заказа можно оставить отзыв.
- Отзывы отображаются на странице собственности с указанием даты бронирования и сортируются по orders.start_date
- TODO: К отзыву можно приктеплять фотографии.
- TODO: Создать таймер, который позволит завершать заказы по окончанию. При завершении заказа деньги переводятся хосту.

- Для сдачи жилья нужно добавить его на свой аккаунт. После одобрения модератором жильё будет доступно для съёма.
- Можно добавить фотографии жилья или изменить их в любое время.
- Жильё и фотографии должны быть одобрены модератором перед добавлением в листинг.
- В любое время владелец может убрать собственность из листинга без удаления и вернуть обратно.
- При получении заказа владелец может принять его или отклонить.

  Администрирование:

- Роль ADMIN существует изначально.
- Роль MODER может создать, приостановить или удалить только ADMIN.
- Модераторы - работники компании, должны подтверждать собственность, картинки и отзывы, а также модерировать ресурс.
